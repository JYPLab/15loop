# 15Loop Product Marketing Context

*Last updated: 2026-07-19*

## Product Overview

**One-liner:** 15Loop는 초등 5·6학년과 중학교 1학년이 영어 단어의 뜻·소리·문맥·떠올리기 중 어디에서 연결이 끊기는지 진단하고, 약한 연결부터 매일 15분 학습하게 하는 부모 연결형 영어 단어 서비스다.

**What it does:** 아이는 가입 없이 20개 단어 진단을 시작하고, 약한 연결이 발견되면 5개 집중 문항을 추가로 푼다. 결과를 본 뒤 부모가 Google 또는 이메일로 가입해 아이 프로필에 결과를 연결하고 7일 동안 매일 15분 적응형 학습을 진행한다. GPT-5.6은 문자열 일치로 판단하기 어려운 제한된 문장·의미 사용을 평가한다.

**Product category:** 초등 고학년·중학교 초반 영어 단어 진단 및 적응형 복습 서비스

**Product type:** B2C 교육 SaaS형 웹서비스. 사용자는 아이이고 계정 소유자·구매자는 부모다.

**Business model:** 오픈 베타 7일 무료, 결제정보 미수집. 초기 가격 가설은 자녀 1명 기준 30일 이용권 12,900원이다. 결제는 오픈 베타와 해커톤 심사 경로에서 제외한다.

## Target Audience

**Primary household:** 자녀가 초등학교 5·6학년 또는 중학교 1학년이고, 선행·해외 체류·영어 노출이 충분하지 않아 단어를 읽고 듣고 문장에서 이해하는 연결이 약하다고 느끼는 한국 가정.

**Decision maker and payer:** 부모 또는 보호자

**End user:** 초등학교 5·6학년 및 중학교 1학년 학습자

**Primary use case:** 아이가 단어 뜻은 외운 것 같지만 발음·듣기·문장·회상에서는 막힐 때, 부모가 어디에서 연결이 끊기는지 확인하고 매일 15분 학습을 이어가게 한다.

**Jobs to be done:**

- 아이가 단어를 정말 아는지, 단순히 한국어 뜻만 외웠는지 확인한다.
- 부족한 연결부터 짧고 반복 가능한 학습으로 보강한다.
- 부모가 매번 옆에서 문제를 내지 않아도 학습일·시간·반복 결과를 확인한다.
- 가격을 지불하기 전에 아이가 실제로 3일 이상 사용하는지 확인한다.

**Important secondary audience:** 15Loop를 만든 과정을 보고 자기 서비스를 출시하고 싶은 바이브 코딩 관심자. 이들은 15Loop의 구매자가 아니라 Project JYP의 잠재 고객이므로 부모 모집 CTA와 같은 게시물에서 섞지 않는다.

| 아이 | 어렵지 않음, 빠른 피드백, 15분 안에 끝남 | 같은 단어를 반복해서 다른 방식으로 답하는 과정이 숙제처럼 느껴질 수 있음 | 한 단어 전체를 길게 쓰게 하지 않고 약한 연결을 짧게 연습 |

| 베타 운영자 | 반복 사용과 가격 의향 | 무료 가입·조회수가 구매 가능성을 과장함 | 3일 완료 학습과 12,900원 의향을 함께 측정 |

## Problems & Pain Points

**Core problem:** 영어 단어장은 철자와 한국어 뜻을 한 페이지에 제공하지만, 학습자가 그 단어를 어떻게 읽는지 반복해서 듣고, 문장에서 이해하고, 필요할 때 떠올리는 과정은 분리돼 있다. 그래서 부모와 아이는 “외웠는데 왜 읽거나 들으면 모르지?”라는 혼란을 겪는다.

**Why alternatives fall short:**

- 종이 단어장은 소리를 듣기 위해 별도의 검색이나 부가 기능이 필요하다.
- 무조건 반복 암기는 어느 연결이 약한지 구분하지 않는다.
- 고정된 하루 단어 수는 아이의 실제 집중시간과 난이도를 반영하지 않는다.
- 무료 체험 전에 가입부터 요구하면 아이가 가치를 경험하기 전에 부모가 이탈한다.
- 공개 랭킹과 과도한 경쟁은 아동 개인정보와 학습 불안을 키울 수 있다.

**What it costs them:** 학원·교재·부모의 관리시간을 쓰고도 실제로 무엇이 부족한지 모른다. 아이는 “나는 영어를 못한다”고 느끼고, 부모는 선행을 시키지 않은 결정에 불안과 죄책감을 느낄 수 있다.

**Emotional tension:** 아이를 압박하고 싶지는 않지만 학교 영어는 따라가야 한다. 부모는 더 많이 외우게 하는 것이 해결책인지 확신하지 못한다.

## Competitive Landscape

현재 경쟁사별 정량 비교는 완료하지 않았다. 마케팅에서는 검증되지 않은 우월성 주장을 하지 않고 다음 대안 범주와의 차이만 설명한다.

**Direct alternatives:** 단어 암기 앱과 온라인 단어 학습 서비스 — 반복과 퀴즈를 제공하지만, 15Loop는 네 연결을 별도로 평가하고 다음 큐에 반영하는 것을 핵심으로 둔다.

**Secondary alternatives:** 영어학원, 과외, 문제집 — 더 넓은 교육을 제공하지만 부모가 집에서 짧게 아이의 단어 연결 상태를 확인하기 어렵다.

**Indirect alternatives:** 종이 단어장, 부모가 직접 문제 내기, 유튜브 영어 노출 — 비용은 낮을 수 있지만 개인별 평가·저장·다음 복습 결정이 자동으로 이어지지 않는다.

## Differentiation

**Key differentiators:**

- 뜻·소리·문맥·떠올리기를 하나의 점수가 아닌 네 연결로 분리한다.
- 아이가 먼저 가입 없이 가치를 경험하고 부모 계정은 결과 뒤에 연결한다.
- 하루 30단어가 아니라 15분 동안 실제 활동한 시간과 연결한 고유 단어 수를 본다.
- 틀린 단어를 즉시 반복하지 않고 다른 단어 뒤에 다시 배치한다.
- AI의 주 역할을 무제한 생성이 아니라 제한된 언어 평가로 둔다.
- 학습 데이터가 쌓일수록 다음 복습 우선순위가 개인별로 달라진다.

**How we do it differently:** 결정론적 커리큘럼·시간·계정·복습 정책 위에 GPT-5.6의 의미 평가를 제한적으로 결합한다. 현재 공개 학습 단어는 검수된 30개로 제한하고, 3,000개 교육과정 지도와 400개 후보 데이터는 검수·발행 게이트 뒤에 둔다.

**Why that's better:** 아이에게 불필요한 쓰기 부담을 줄이고, 부모에게 “몇 개 외웠다”보다 “어느 연결을 먼저 보강해야 한다”는 다음 행동을 준다.

## Objections

| 반론 | 답변 |
|---|---|
| 단어 30개로 실력을 알 수 있나? | 학교 성적이나 전체 실력을 판정하는 시험이 아니다. 현재 검수된 풀에서 연결 패턴을 확인하는 베타 진단이며 범위는 검수 후 확장한다. |
| AI가 아이를 잘못 평가하면 어떡하나? | AI는 제한된 선택 과제의 선택 정답을 결정하지 않는다. 서버가 아는 커리큘럼 안에서 의미 평가가 필요한 선택적 문장 과제만 처리하며 출처를 표시하고 로컬 대체 평가도 구분한다. |
| 결국 또 외우는 앱 아닌가? | 반복 자체는 필요하지만, 동일 반복이 아니라 약한 연결·기한·오답 이력에 따라 다시 만나는 순서를 바꾼다. |
| 7일 만에 성적이 오르나? | 성적 향상을 약속하지 않는다. 7일은 아이가 15분 학습을 반복할 수 있는지와 부모가 가치를 느끼는지 확인하는 베타 기간이다. |
| 아이가 직접 가입해야 하나? | 무료 진단은 무가입이고, 계속 학습 기록은 부모 계정 아래 아이 프로필에 저장한다. |

**Anti-persona:** 현재 범위 밖 학년, 단기간 성적 보장만 원하는 가정, 자녀 대신 부모가 답을 입력하려는 가정, 무료 자료만 받고 실제 사용·피드백 의사가 없는 가정.

## Switching Dynamics

**Push:** 뜻은 외웠는데 읽거나 들으면 모르는 반복 경험, 학원이나 교재를 써도 정확한 약점을 모르는 답답함.

**Pull:** 가입 없는 무료 진단, 부모 친화적인 결과 언어, 매일 15분, 가격 사전 공개.

**Habit:** 기존 단어장·학원 숙제·하루 단어 수 목표가 익숙하고, “영어는 무조건 외워야 한다”는 믿음.

**Anxiety:** 단어 수가 적어 보임, AI 평가 신뢰, 아이 개인정보, 7일 이후 결제, 아이가 다시 사용하지 않을 가능성.

## Customer Language

아래 문장은 창업자 본인의 문제 인식과 가정 내 관찰에서 나온 표현이다. 외부 고객 인터뷰 인용으로 표시하지 않는다.

**Problem language:**

- “책은 단어가 무슨 뜻인지만 나와 있잖아.”
- “어떻게 읽는지는 또 찾아야 하잖아.”
- “언어 학습이라는 게 계속 듣는 게 기본인데, 여기서부터 이슈가 있지 않나.”
- “중학교 영단어도 아직 잘 모른다는 생각이 들어.”
- “무조건 외우는 게 답이라고 하지만 현실적으로 정말 어렵다.”
- “단어를 보고 정확히 듣고 상황을 인지해가면서 익히는 것이라고 생각한다.”
- “결론은 evaluation이잖아.”
- “사람의 기억은 반복 학습을 통해 오래 기억되지만, 랜덤하게 반복할 수 있어야 한다.”

**Founder story:**

- “선행을 안 시킨 아빠가 중학교에 간 아이의 영어 성적을 고민하다가 만들었다.”

**Words to use:** 연결, 약한 연결, 떠올리기, 매일 15분, 가입 없이 진단, 부모에게 연결, 실제 학습일, 검수된 단어, 다음에 무엇을 복습할지.

**Words to avoid:** 인출(부모 대상 카피), 완벽 암기, 성적 보장, AI가 자동으로 실력을 향상, 7일 만에 향상, 무조건 맞춤, 검증되지 않은 학원비·시장 수치.

**Glossary:**

| 용어 | 의미 |
|---|---|
| 네 연결 | 보고 의미 연결, 듣고 단어 연결, 문맥에서 이해, 뜻 보고 단어 떠올리기 |
| 연결 단어 수 | 15분 동안 학습 사이클을 완료한 서로 다른 단어 수 |
| 완료 학습일 | 15분 세션 완료 이벤트가 기록된 서로 다른 날짜 수 |
| WPA | 3일 이상 실제 학습하고 12,900원 결제 검토 의향을 밝힌 가격 검증 가정 |
| 로컬 평가 | OpenAI 경로를 쓸 수 없을 때 작동하며 화면에 별도로 표시되는 결정론적 대체 평가 |

## Brand Voice

**Tone:** 솔직한 부모의 관찰, 차분한 문제 해결, 과장 없는 실험 공개.

**Style:** 짧고 구체적이며 생활 장면으로 시작한다. 전문용어보다 부모와 아이가 실제로 쓰는 말을 사용한다.

**Personality:** 현실적, 다정함, 집요함, 투명함, 실험적.

## Proof Points

학습 효과 수치는 아직 증명되지 않았다. 현재 공개 가능한 증거는 제품과 실행 증거다.

**Build evidence:**

- 2026년 7월 15일부터 19일까지 5일 동안 공개 저장소에 36개 커밋
- 자동 테스트 42개 통과
- D1 마이그레이션 8개, 데이터 테이블 13개, API 라우트 11개
- 검수된 학습 단어 30개
- 교육부 2022 개정 기본 어휘 지도 3,000개
- 비공개 후보 카탈로그: 중학교 핵심 300개, 일상 표현 100개
- 실제 운영 도메인과 Google·이메일 부모 로그인 흐름
- 모바일, 검색, 분석, 개인정보 경로를 포함한 오픈베타 제품

**Claims not yet available:** 학습 점수 향상, 기억 유지율, 유료 전환율, 고객 후기, 7일 전후 비교, 발음 개선.

## Goals

**Primary business goal:** 첫 오픈베타에서 실제로 반복 사용하는 가정과 가격을 지불할 가능성이 있는 부모를 찾는다.

**Key conversion action:** 아이의 무료 진단 완료 → 부모 계정 연결 → 15분 학습을 서로 다른 3일 완료 → 12,900원 가격 의향 응답.

**4-week target:** 가격을 미리 본 적합 부모 30가정, 진단 완료 20가정, 첫 학습 15가정, 3일 반복 10가정, 7일 완주 6가정, WPA 5가정.

**Content goal:** 부모에게는 무료 진단을 시작하게 하고, 바이브 코딩 오디언스에게는 15Loop 개발기를 통해 Project JYP의 프로젝트 브리프 제출과 첫 빌드 행동을 만들게 한다.

## Context Boundaries

- 15Loop 제품 KPI와 Project JYP 콘텐츠 KPI를 합치지 않는다.
- “베타 1,000명”은 공개 연재용 도전 목표일 수 있지만 15Loop의 제품 검증 성공 기준은 아니다.
- 부모 대상 게시물에는 무료 진단 CTA 하나를 사용한다.
- 빌더 대상 게시물에는 개발기·브리프 CTA 하나를 사용한다.
- 같은 게시물에서 부모 모집과 바이브 코딩 서비스 판매를 동시에 요구하지 않는다.
