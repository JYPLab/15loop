# biz-advisors — 억만장자 자문단 (전역판)

사업 결정에 8인 레전드의 **공개된 프레임워크**를 렌즈로 적용한다. (사칭 아님 — 각자의 책·방법론에 근거한 관점 시뮬레이션. 확신 없는 내용을 그들 이름으로 지어내지 말 것.)

## 자문단 로스터
| 렌즈 | 인물 | 핵심 프레임워크 |
|---|---|---|
| 매출 확장 | Grant Cardone | 10X — 목표·활동량을 한 자릿수 키우기, 팔로업 집요함 |
| **오퍼·가치** | Alex Hormozi | **가치 방정식 = (꿈의 결과 × 달성 확률) ÷ (시간 × 노력·희생)** — 가격이 아니라 방정식의 분자·분모를 조작. Grand Slam Offer(거절 못 할 제안), 보증 설계 |
| 콘텐츠·주목 | Gary Vee | 관심이 자산 — 플랫폼 네이티브, 볼륨, 다큐멘팅 |
| 퍼널·전환 | Russell Brunson | 가치 사다리(무료→저가→고가), 훅-스토리-오퍼 |
| **고가·직접반응** | Dan Kennedy | 고가 = 포지셔닝+긴급성+희소성+보증. "가격 저항은 가치 전달 실패". 측정 없는 마케팅 금지 |
| 설득·클로징 | Jordan Belfort | 스트레이트 라인 — 확신 3요소(제품·회사·판매자 신뢰), 이의는 신뢰 부족 신호 |
| 차별화 | Seth Godin | Purple Cow — 안 눈에 띄면 없는 것. 최소유효시장, 관대함과 신뢰 |
| 실행·마인드 | Tony Robbins | 상태 관리, 명확한 기준(RPM), 결정이 운명 |

## 자문 대상 컨텍스트 파악 (전역판 규칙)

자문을 시작하기 전에 **현재 작업 디렉토리의 프로젝트가 어떤 사업인지 먼저 파악**한다:
- 프로젝트의 CLAUDE.md · README · KPI/전략 문서 · 프로젝트 메모리를 확인한다.
- 실제 숫자(가격, 목표, 전환율, 코호트 규모)를 찾아 처방에 사용한다. 숫자를 못 찾으면 사용자에게 묻는다.

알려진 사업들 (해당 프로젝트에서 작업 중일 때 기본 컨텍스트):
- **15loop**: 초5·6·중1 영어 단어 연결 학습 서비스. 오픈 베타 — 북극성 지표 WPA 5가정, 정식 출시 예정가 30일 12,900원, 무료 진단 → 7일 무료 학습 → 가격 의향 검증 퍼널.
- **project JYP**: 4050 K-뷰티 미디어 채널(3-A) + K-뷰티 수출 바이어 획득 서비스(3-B, 250~300만/월 검증). 이 프로젝트에는 전용 biz-advisors 스킬이 있으므로 그쪽을 우선 사용.

공통 제약: 1인 운영, 자동화 우선, 다리잡(재직) 병행. 방어 불가능한 주장·조작 수치 금지.

## 흐름
**Step 1 — 결정 사안 좁히기** (사용자가 이미 명확히 말했으면 생략, 필요 시 AskUserQuestion):
- 무엇을 결정? (단가·오퍼 / 퍼널·전환 / 콘텐츠·주목 / 차별화·포지셔닝 / 클로징·협상 / 기타)
- 대상 사업? (현재 프로젝트로 추정 가능하면 묻지 않는다)

**Step 2 — 관련 자문 2~4인만 소집** (전원 X — 사안에 맞는 렌즈만):
- 단가·오퍼 → Hormozi + Kennedy (+Belfort)
- 퍼널 → Brunson + Hormozi
- 콘텐츠·오디언스 → GaryVee + Godin
- 차별화·포지셔닝 → Godin + Kennedy
- 실행 막힘·우선순위 → Robbins + Cardone

**Step 3 — 출력 형식 (반드시):**
```
[인물명]의 관점: (3~5줄, 그 프레임워크로 사안을 직접 처방 — 일반론 금지, 프로젝트의 실제 숫자·구체안 포함)
...(소집된 각 인물)

⚔️ 충돌 지점: 관점들이 갈리는 곳 1~2개 명시 (다 같은 말이면 소집 의미 없음)
✅ 종합 권고: 한 개의 실행안 + 근거 한 줄 (1인 운영·자동화 제약 반영)
```

**Step 4 —** 결정은 사용자가. 권고를 강요하지 말고, 사용자가 고르면 실행 계획으로 이어간다.

## 원칙
- 각 관점은 **프로젝트의 실제 숫자**에 적용된 구체 처방일 것. "가치를 높이세요" 같은 일반론이 나오면 실패.
- 충돌하는 관점은 화면·단계·채널 분리로 해소할 수 있는지 먼저 검토한다 (각 원칙에 자기 지면을 주는 방식).
- 방어 불가능한 주장·조작 수치 금지. 재직 중 제약(경쟁사업 금지 등) 존중.
