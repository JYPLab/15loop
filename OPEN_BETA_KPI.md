# 15Loop — 초5·6·중1 오픈 베타 KPI

> **정본 안내 (2026-07-19):** 전체 운영 정본은 `docs/handoff/04_OPEN_BETA_OPERATIONS_HANDOFF.md`다. 현재 구현은 WPA 자격을 `서로 다른 3일의 15분 학습 완료`로 판정하며, 이 문서 아래의 `총 45분 이상` 대체 조건은 사용하지 않는다. D0↔D7 점수 변화는 향후 측정 후보이며 현재 제품 약속이 아니다.

> 작성일: 2026-07-18
> 목표: 무료 사용자를 많이 모으는 것이 아니라, 자녀의 영어 문제를 실제로 해결하고 싶고 효과가 확인되면 돈을 낼 부모를 찾는다.

## 1. 모집 대상

### 적합 가정

- 자녀가 초등 5·6학년 또는 중학교 1학년이다.
- 단어의 뜻뿐 아니라 읽기·듣기·문장 사용에 어려움이 있다.
- 보호자가 7일 동안 최소 세 번 학습 여부를 확인할 수 있다.
- 서비스가 도움이 되면 30일 12,900원 수준의 결제를 검토할 의향이 있다.

### 모집에서 제외

- 가격을 알리지 않은 무료 체험 모집
- 학년이 현재 베타 범위 밖인 가정
- 자녀 대신 보호자가 답을 입력하려는 경우
- 무료 자료만 받고 사용·인터뷰 의사가 없는 경우

`돈을 낼 부모`는 소득 수준으로 판정하지 않는다. 가격을 미리 본 뒤 사용 행동과 유료 의향을 함께 보인 부모로 정의한다.

## 2. 북극성 지표

### 유료의향 활성가정(WPA, Willing-to-Pay Active Household)

아래 조건을 모두 충족한 고유 가정 수다.

1. 부모가 30일 예상가격 12,900원을 모집 전에 확인했다.
2. 자녀가 무료 진단을 완료하고 부모 계정에 연결했다.
3. 7일 안에 서로 다른 날 3일 이상 학습하거나 총 45분 이상 학습했다.
4. 부모가 `12,900원에 출시되면 결제를 검토하겠다`고 선택했다.

결제가 열리기 전까지는 이를 **구매자**가 아니라 **가격 검증 리드**라고 부른다.

**4주 목표: WPA 5가정**

## 3. 4주 KPI 퍼널

| 단계 | KPI | 4주 목표 | 계산 방식 |
|---|---:|---:|---|
| 모집 | 가격을 본 적합 부모 신청 | 30가정 | 학년·문제·가격 의향 확인 |
| 진단 | 무료 진단 완료 | 20가정 | 진단 결과 저장 완료 |
| 계정 | 진단을 자녀 프로필에 연결 | 18가정 | `diagnostic_claimed` |
| 첫 가치 | 첫 학습 완료 | 15가정 | 첫 평가 이벤트 저장 |
| 반복 | 3일 이상 학습 | 10가정 | 서로 다른 학습일 3일 이상 |
| 완주 | 7일 프로그램 완료 | 6가정 | 7일 중 5일 또는 총 75분 이상 |
| 가격 검증 | WPA | 5가정 | 행동 조건 + 12,900원 의향 |
| 인터뷰 | 부모 인터뷰 완료 | 5가정 | 15분 인터뷰 기록 완료 |

목표치는 첫 코호트의 검증 기준이며 시장 규모 예측이 아니다.

## 4. 핵심 전환율

| 전환율 | 1차 기준 | 의미 |
|---|---:|---|
| 적합 신청 → 진단 완료 | 65% 이상 | 모집 메시지와 첫 경험이 맞는가 |
| 진단 완료 → 프로필 연결 | 80% 이상 | 부모 가입 허들이 적절한가 |
| 프로필 연결 → 첫 학습 | 80% 이상 | 진단 결과가 다음 행동을 만드는가 |
| 첫 학습 → 3일 반복 | 60% 이상 | 아이가 다시 사용할 이유가 있는가 |
| 3일 반복 → 7일 완주 | 50% 이상 | 15분 루프가 지속 가능한가 |
| 3일 반복 → WPA | 40% 이상 | 사용 가치가 가격 의향으로 이어지는가 |

## 5. 제품·학습 KPI

### 반드시 보는 지표

- 가정별 활성 학습일 수
- 총 학습시간과 하루 평균 학습시간
- 하루에 연결한 고유 단어 수
- 보기·듣기·문맥·인출별 최초 점수와 7일차 점수 변화
- GPT 평가 성공률과 로컬 평가 전환율
- 다시 출제된 단어의 정답률 변화
- 부모가 관찰한 아이의 자발적 재사용 여부

### 보조지표

- 방문자 수
- 무료 진단 시작 수
- 가입자 수
- 친구 챌린지 공유 수

보조지표가 늘어도 3일 반복과 WPA가 늘지 않으면 성공으로 보지 않는다.

## 6. 필요한 측정 이벤트

### 현재 이미 있는 이벤트

- `guardian_consent_accepted`
- `learner_created`
- `diagnostic_claimed`
- `daily_session_completed`

### 모집 전에 추가할 이벤트

- `beta_application_submitted`
- `diagnosis_started`
- `diagnosis_completed`
- `learning_session_started`
- `ai_evaluation_completed`
- `price_intent_submitted`
- `parent_interview_completed`

공통 속성:

- `guardian_id` 또는 익명 내부 ID
- `learner_grade`
- `cohort_id`
- `utm_source`, `utm_medium`, `utm_campaign`, `utm_content`
- `evaluation_source` (`openai`, `local-fallback`)

아이의 실명·학교·연락처·자유서술 답변을 분석 이벤트에 넣지 않는다.

## 7. 유료 의향 질문

가격을 숨긴 채 `유료로 쓰겠습니까?`라고 묻지 않는다. 3일차와 7일차에 동일하게 묻는다.

> 15Loop가 지금처럼 아이의 약한 연결을 찾아 매일 15분 학습을 제공한다면, 자녀 1명 기준 30일 12,900원에 이용하시겠어요?

- 출시되면 결제를 검토하겠다
- 개선되는 모습을 더 확인하고 싶다
- 이 가격에는 이용하지 않겠다

이어 한 가지만 묻는다.

> 그렇게 선택한 가장 큰 이유는 무엇인가요?

## 8. 코호트와 채널

첫 코호트는 30가정 이내로 제한하고 모든 모집 링크에 UTM을 사용한다.

```text
utm_source=parent_community|instagram|threads|referral|direct
utm_medium=community_post|organic_social|dm|referral
utm_campaign=15loop_beta_2026q3
utm_content={post_or_partner_id}
```

채널 평가는 신청 수가 아니라 `채널별 WPA 수 / 적합 신청 수`로 한다.

## 9. 주간 판정

매주 아래를 한 표로 본다.

| 항목 | 이번 주 | 누적 | 목표 | 조치 |
|---|---:|---:|---:|---|
| 가격을 본 적합 신청 |  |  | 30 |  |
| 진단 완료 |  |  | 20 |  |
| 첫 학습 |  |  | 15 |  |
| 3일 반복 |  |  | 10 |  |
| 7일 완주 |  |  | 6 |  |
| WPA |  |  | 5 |  |
| 부모 인터뷰 |  |  | 5 |  |

### GO 기준

- WPA 5가정 이상
- 첫 학습 → 3일 반복 전환율 60% 이상
- 7일 완주 가정의 절반 이상이 점수 변화 또는 학습 편의성을 체감
- GPT 평가 성공률 95% 이상

### 수정 기준

- 진단 이탈이 크다 → 문항 수·설명·진단 결과의 즉시가치 수정
- 가입 이탈이 크다 → 부모 가입과 자녀 연결 단계 수정
- 첫날만 쓰고 돌아오지 않는다 → 단어 난이도·재방문 장치·부모 리마인드 수정
- 사용은 하지만 유료 의향이 없다 → 부모 리포트·효과 증거·가격 가설 수정
- 유료 의향은 있지만 아이가 쓰지 않는다 → 구매자가 아닌 응원성 응답으로 분류

## 10. 이번 주 실행

1. 모집 문구에 `초5·6·중1`, `7일`, `향후 예상가격 12,900원`을 함께 공개한다.
2. 학년·현재 문제·가격 의향을 묻는 부모 신청 폼을 만든다.
3. 10가정씩 세 번 초대하고 코호트를 분리한다.
4. 3일차 가격 의향 질문과 7일차 인터뷰 일정을 예약한다.
5. 첫 코호트가 끝날 때까지 광고비보다 직접 모집과 추천을 우선한다.
